# 中小企業診断士 過去問学習アプリ

中小企業診断士試験(1次・2次)の過去問学習 PWA。2015〜2025年度の全過去問
(1次: 84試験 2,345問 / 2次: 44事例 181問)を収録し、完全オフラインで動作する。

仕様は `REQUIREMENTS.md`(唯一の正)、作業規範は `CLAUDE.md` を参照。

## 機能

- **演習モード**: 科目→年度 / 論点タグで選択、1問ずつ即時正誤判定
- **SM-2 間隔反復**: 自己評価から次回出題日を算出、「今日の復習」を提示
- **模試モード**: 本番同一構成+制限時間タイマー、採点(全員正解/採点対象外対応)・合否判定
- **弱点ヒートマップ**: 論点タグ×正答率、タップでその論点の演習へ
- **学習統計**: 日別解答数、科目別正答率、試験日カウントダウン
- **2次学習ビュー**: 与件文/設問表示、字数カウンタ付き答案下書き(端末内保存)
- **全文検索**: 問題文・選択肢・タグ・与件文

## 技術スタック

React 18 + Vite + TypeScript (strict) / Zustand / Dexie (IndexedDB) /
Tailwind CSS / vite-plugin-pwa(全データプリキャッシュ、オフライン動作)

## 開発

```bash
npm install
npm run dev        # 開発サーバ (data/ を public/data/ へ同期してから起動)
npm run build      # 型チェック + 本番ビルド
npm run test       # ユニットテスト (採点ロジック・SM-2)
```

## ディレクトリ

```
data/       過去問マスターデータ(正本・読み取り専用)
scripts/    データ再生成パイプライン(Python)+ sync-data.mjs
src/
  types/    データ・学習履歴のスキーマ型
  db/       Dexie スキーマ
  data/     ローダー(初回 IndexedDB 取込)・クエリ
  lib/      SM-2 / 採点エンジン / JST日付ユーティリティ
  pages/    画面(演習・模試・統計・2次・検索)
```

## 図表画像について

図表問題(468問)の頁画像は原本PDFから `python3 scripts/extract_figures.py` で
生成する(`public/figures/` に配置)。未生成の場合、アプリは原本PDFへのリンクで代替する。
